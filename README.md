# Crypto Dashboard

A real-time cryptocurrency dashboard web application that lets users track live prices, trends, market data, and more using crypto API data. It provides an intuitive UI to monitor top coins, search for crypto tokens, and view detailed charts and info.


## Features

- View real-time market prices for major cryptocurrencies
- Search for any coin and view its details
- Price charts and trends
- Responsive UI with modern layout

## Installation

1. Clone the repository
```bash
git clone https://github.com/sharmila-as/Crypto_Dashboard.git
cd Crypto_Dashboard
```

2. Setup the frontend
```bash
cd client
npm install
```

Replace .env.example or add your API keys if needed:
```bash
ANGULAR_APP_API_KEY=your_api_key_here
```

3. Run the frontend
```bash
npm start
```
